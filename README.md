# DAuM
Code and dataset of our paper "[Enhanced Aspect Level Sentiment Classification with Auxiliary Memory](https://www.aclweb.org/anthology/C18-1092)" accepted by COLING 2018.

# Requirements
* python 3.6
* tensorflow 1.3.0
* numpy 1.16.4
* scikit-learn 0.21.2
